# Proiect PAAM - Timișoara Events App

Aplicație mobilă și backend pentru evenimente în Timișoara.

## Structura Proiectului

- **backend/**: Server Node.js + Express + MongoDB. Se ocupă de API și de scraping-ul evenimentelor.
- **mobile-app/**: Aplicație React Native (Expo).
- **frontend/**: (Placeholder pentru viitoare dezvoltări web)

## Setup

### Backend
1. Navighează în folderul `backend`.
2. Instalează dependențele: `npm install`.
3. Configurează fișierul `.env` cu string-ul de conexiune MongoDB.
4. Rulează serverul: `npm start`.

### Mobile App
1. Navighează în folderul `mobile-app`.
2. Instalează dependențele: `npm install`.
3. Pornește aplicația: `npx expo start`.
